Thoughts on Web Accessibility

Web Accessibility is when a webpage is made to be accessed by anybody. Whether that be to give audio queues to the visually impaired or visual queues to the auditory impaired. It is mainly used for those with impairments like visual and audio. However, it can also be used for people with normal vision/audio or healthy people. Not to mention that it will also enhance the user experience and allow for wide adaptability.

These kinds of strides are what I feel make the world a better place but also will help create a brighter future. A future that will blur the line of the capable and uncapable.

Importance of Web Accessibility

Inclusive: it allows for everyone to access information

Legal: the government made sure that it was a necessity to be able to make webpages accessible to everybody

User Experience: Accessibility settings will help all users.

Broad audience: not only those that are in need but older people can also be able to access the webpage a little better with these accessability features.
